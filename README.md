
## API
```
python3 app.py
0.0.0.0:5000/users?id={}  ==> search by id
0.0.0.0:5000/users?first_name={} ==> search by first_name
0.0.0.0:5000/users?last_name={} ==> search by last_name
0.0.0.0:5000/users?email={} ==> search by email
0.0.0.0:5000/users?gender={} ==> search by gender
0.0.0.0:5000/users?age={} ==> search by age
0.0.0.0:5000/insert?id={}&first_name={}&last_name={}&email={}&gender={}&age={} 
0.0.0.0:5000/update?id={}&first_name={}&last_name={}&email={}&gender={}&age={}
0.0.0.0:5000/delete?id={}  ==> delete user by id
```
